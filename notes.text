What is Redux?
-> Redux is a state management library for JavaScript apps, often used with React.

Difference between React and Redux?
-> React is for building user interfaces, while Redux manages application state.

What does Redux store?
-> Redux stores the state of the entire application in a single object called the store.

What are actions in Redux?
-> Actions are objects that describe events that happened in the application.

What is a reducer in Redux?
-> A reducer is a pure function that takes the current state and an action, then returns the new state.

What is the store in Redux?
-> The store is an object that holds the application's state and allows access to it via functions like `getState()`.

What is the dispatch function in Redux?
-> Dispatch sends actions to the Redux store, triggering the reducer to update the state.

What is the role of getState() in Redux?
-> `getState()` retrieves the current state from the Redux store.

What are middleware in Redux?
-> Middleware allows you to extend Redux with custom logic, like handling asynchronous actions.

What is combineReducers in Redux?
-> `combineReducers` merges multiple reducers into a single reducer function.

What is Redux Thunk?
-> Redux Thunk is middleware that lets you write action creators that return functions instead of actions.

What are selectors in Redux?
-> Selectors are functions that extract and compute derived data from the Redux store.

What is the Provider component in Redux?
-> `Provider` makes the Redux store available to all components in the app.

What is mapStateToProps in Redux?
-> `mapStateToProps` connects a component to specific parts of the Redux store's state.

What is mapDispatchToProps in Redux?
-> `mapDispatchToProps` connects a component to the Redux store's dispatch function.

What are Redux DevTools?
-> Redux DevTools help you inspect every action and state change in your application.

Why is immutable state important in Redux?
-> Immutable state ensures that state changes are predictable and easy to track.

What is the connect function in Redux?
-> `connect` is a function that links React components to the Redux store.

What does Redux Toolkit do?
-> Redux Toolkit simplifies the setup of Redux with pre-configured tools and best practices.

How does Redux handle side effects?
-> Redux handles side effects using middleware like Redux Thunk or Redux Saga.
